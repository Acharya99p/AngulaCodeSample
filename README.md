# AngularExercise

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.2.5.

## Installation

Run `npm install` to install all the dependencies for the project.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Build

Run `ng build` to build the project. The build artifacts will be stored in the `dist/` directory. Use the `--prod` flag for a production build.

## Running unit tests

Run `ng test` to execute the unit tests via [Karma](https://karma-runner.github.io).
Run `ng test --code-coverage` to run the tests and find the code coverage.

## End Notes
### Email
* Email validation is carried out using HTML5's and Angular's own email validation as the requirement did not specify any pattern for the email.
* Hence "a@a" is also a valid email address.

### Password
The below mentioned requirements are carried out:-
* Must be 8 characters long.
* Must have a character.
* Must have a special character.
